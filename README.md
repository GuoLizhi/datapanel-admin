# beastadmin

## 高频交易策略管理后台前端仓库

技术选型：React + Ant Design

## 启动项目

开发环境：http://127.0.0.1:5173/

```bash
yarn
npm run dev
```

## 项目结构

```
- assets      静态资源文件夹
- common      公共方法文件夹
- components  公共组件文件夹
- hooks       公共hooks文件夹
- locales     i18n文件夹
- pages       页面文件夹
- routes      路由文件夹
- service     请求文件夹
- store       全局状态文件夹
```
